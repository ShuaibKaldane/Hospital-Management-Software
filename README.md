# 🏥 Hospital Management System

A full-stack Hospital Management System built using **Angular**, **Node.js**, **Express.js**, and **MySQL**. The application provides secure role-based access for **Admin**, **Doctor**, and **Patient**, allowing efficient management of appointments, prescriptions, patient records, doctor salaries, and payments.

This project is being developed incrementally using the **Agile Scrum methodology**, with each sprint focusing on a specific set of features.

---

# 📌 Project Objective

The objective of this project is to build a real-world hospital management application while learning professional full-stack development practices.

The project focuses on:

- Angular Frontend Development
- REST API Development
